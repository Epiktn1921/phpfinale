# Интернет-магазин книг (e-shop)

Учебный проект интернет-магазина с админкой, корзиной и заказами.

## Технологии
- PHP 8+
- MySQL
- Хранимые процедуры
- Куки (корзина), сессии (авторизация)

## Установка
1. Создайте БД `eshop`
2. Импортируйте `database_schema.sql`
3. Настройте `config.php`
4. Запустите через OpenServer, XAMPP или `php -S localhost:8000`

## Вход в админку
- URL: `/enter.php`
- Логин: `admin`
- Пароль: `admin123`

> Все данные хранятся локально. Проект не требует Composer.
